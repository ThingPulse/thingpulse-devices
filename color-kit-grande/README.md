# Color Kit Grande

The Color Kit Grande is a versatile ESP32-based development board designed for creating colorful IoT projects and home automation applications.

## Features

- ESP32-based microcontroller
- Full-color display
- Compatible with ESPHome
- Ready for home automation integration
- Easy to program and customize

## Configuration

This repository includes ESPHome configuration files to help you get started:
- `esphome/home-automation.yaml` - Home automation setup
- `esphome/watch.yaml` - Watch/display configuration
- `esphome/secrets.yaml` - Configuration secrets (customize for your setup)
